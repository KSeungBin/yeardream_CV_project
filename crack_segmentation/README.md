## 3. Setup
---
To run this project, install it locally:
```
$ pip install -U Werkzeug
$ pip install natsort
$ pip install ipywidgets (or 'conda install -c conda-forge ipywidgets')
$ pip install pytorch-lightning
$ pip install segmentation-models-pytorch
```
